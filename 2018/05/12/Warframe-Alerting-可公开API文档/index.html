    游戏信息获取


    本章节介绍的API来源于WarframeStat，并非是由我开发的，这里介绍的API有限，如果想获取更多，请自行前往该网站。
    获取全部信息

    请求地址: https://api.warframestat.us/{对应平台}

    请求方式: GET

    返回结果:

    • 对应平台指的是游戏平台，即：PC/PS4/XB1
    • 该请求将返回包括希图斯、奸商、新闻、警报、入侵等所有可公布的信息，信息量巨大。如果你需要的是某个单一条目，请避免使用此API

    集团任务 API

    请求地址: https://api.warframestat.us/{对应平台}/syndicateMissions

    请求方式: GET

    返回结果:

    • 该API唯一有价值的就是希图斯的Ostrons赏金任务。其它集团的任务仅提供地点而没有任务类型(jobs字段形同虚设),且因集团任务往往和该节点本身任务不匹配，所以其它集团的信息没有什么价值。
    • 赏金任务在整个返回列表的第十位(最后一个),jobs内包含5个常规赏金任务。赏金任务内含rewardPool字段，其中包含了赏金奖励列表。

    WM价格查询


    该接口未经WM方允许，请不要用做商业用途。
    接口请求时，请注意url中段的条目名，中文开发者请务必使用我提供的词库中的WF_Sale表进行查询，我已将对应的词条查询码整合，search字段就是。
    另外，该API貌似只支持PC物品查询

    请求地址: https://api.warframe.market/v1/items/{条目名}/orders?include=item

    请求方式: GET

    返回结果:


    由于国内的特殊原因，WM接口请求可能会有卡顿，甚至请求无响应。所以我建议有条件的同学最好使用服务器对该API进行二次包装，进行一次中转(我就是这么做的)。
    另外有一个坑，请使用.NET开发并托管在IIS服务器上的同学注意，WM的TLS握手协议为1.2版本，为稳妥，请在请求函数中加上这样一条代码: System.Net.ServicePointManager.SecurityProtocol = System.Net.SecurityProtocolType.Tls11 | System.Net.SecurityProtocolType.Tls12;
    否则你可能会获取不到数据

    词库下载

    所有的数据是开发者(也就是本人)费老鼻子劲整理的，近一个月时间起早贪黑编撰数据库，并在后期不断进行更新维护。在此感谢Warframe 灰机Wiki和WARFRAME WIKI的数据整理


    由于词库的特殊性，未来会随着游戏的更新持续更新，如果其中未包含你需要的词条，请在Github上issue一下哟~也请持续关注我的Github主页，及时更新新版本

    目前数据库已经基本定型，经过WFA软件半年多的持续性维护更新，词条也收集的相对齐全，在此公开以下几个数据表：

    1. WF_Dict 词典表，一份相对齐全的WF中英对照表(有Id,Type,Zh,En四个字段)

    2. WF_Sale 交易物品表，内含绝大部分可交易物品(其中的Search字段为WM专属查询码)

    3. WF_Alert 警报奖励物品表(囊括警报中可能出现的奖励，相比词典，多了一些头盔等特殊奖励)

    4. WF_Invasion 入侵奖励物品表(囊括入侵中可能出现的奖励，相比词典，多了亡魂、破坏者等武器部件)
