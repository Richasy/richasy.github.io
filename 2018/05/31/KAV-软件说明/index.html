    Key & Value 桌面备忘录

    一种新的方式，记录你的所思所想，记录你的个人数据，将你散落在各处的文件统一整合，如果你需要这些功能，它会是你最好的选择

    软件简介

    这款软件的根本目的在于解放你的大脑，拓展你的思路。
    这是一款很有意思的软件，可以将其看作备忘录和文件索引器的结合。本文将简要介绍这款软件在这两个方面的独特优势，以为你是否选择这款软件做一个参考

    接下来的内容就是叙述这个软件的具体功能，但有些话要说在前面：

    自由分类

    软件最初集成了3个公共分类：全部条目,加星重点和临时列表，并默认添加一个用户分类：默认列表，默认列表内自动添加了开发者博客（网页类型）的中英双语版本

    在标签列表，通过点击+号，你可以添加标签组，自由设置名称,标志和重要级，软件默认提供了30种标志备选：

    设置好标签后，你如果想更改标签的内容，可以在标签上右键单击 （ Windows平板用户是长按 ）

    条目操作

    在选中标签之后（必须是用户标签才能执行条目添加的操作）

    点击条目列表上部，搜索建议框旁边的+按钮，可以填加一个自定义类型的条目：


    例子中展示了文本类型密码和文件类型图片的添加实例，软件共提供10种不同的记录类型，包括：

    • 文本
    • 密码
    • 电话
    • 文件
    • 图片
    • 地点（借助Bing地图记录地点）
    • 音乐
    • 提醒（通过后台在指定时间之前提醒你相关内容）
    • 网页
    • 文件夹

    这些类型都有着各自专属的添加对话框和详情页，我也根据各个类型可能的用途进行了功能的个性化定制，这些你可以自己探索。

    文本类型用于备忘录这一类的功能，需要一个个添加。但是文件类型作为文件索引功能的基石，一个个添加未免太过费力，有没有更好的添加方法呢？

    当然有，批量添加文件的方式就是 拖拽

    类比于你拖拽文件到QQ聊天窗口，你可以将文件批量选取，拖拽进软件右侧的空白区，你所拖拽的所有文件都会被读取并放置在临时列表中

    必须要说明的是，临时列表并不是一个用来存放数据的列表，在关闭软件后，临时列表将会被清空，所以你必须要进入管理模式，将这些添加的条目进行分组

    进入管理模式的方法，就是在之前的添加按钮旁边，有一个扳手图标，点一下，选择管理即可

    选中条目之后点击换个分组，选择好分组，一次批量导入就完成了。但这种方式引入的文件是自动生成的，名字是你的文件名，无备注，所以在引入后你仍需要手动去修改这些条目的数据。

    我想修改条目的内容，如何处理？

    如果你只是想改变条目的名称或者备注，不对条目数据本身进行修改的话，可以在条目上右键单击（ Windows平板用户是长按 ）

    如果你想改变具体的内容，请在页面详情页找到按钮组中的铅笔标签，点击即可修改。

    除了修改条目按钮，你还能看到其它3个按钮，它们的作用是（从左往右）：

    • 黄色：加星重点标注
    • 蓝色：条目修改
    • 粉色：该类型的多功能按钮，包含了常用操作
    • 黑色：复制按钮，根据不同的类型有不同的复制选项

    备份设置

    辛辛苦苦记录的数据如果哪天丢失了那就太让人心痛了。为此，我特别做了两种备份方式本地备份和漫游备份
    默认情况下，软件将以7天的时间间隔漫游你的数据，保存在你的微软账户云端数据中，如果你换了电脑或者其它的情况，可以随时再导入回来。
    但是文件类型的数据，由于UWP的限制，在你卸载软件并重新导入数据后，你需要再用文件选择器选取一次，对软件授予访问该文件的权限。

    安全设置及其它

    如果你需要对软件进行加密打开（默认不加密），可以在其他设置中找到加密方式。提供以下3个选项：

    • 无（即不加密）
    • 密码（打开软件需要输入密码）
    • Windows Hello（采用和你登入计算机相同的验证策略：PIN、指纹或面部识别）

    我个人推荐，能用Windows Hello的尽量使用Windows Hello

    其它设置中，软件支持中英双语言（仅限界面语言）

    除此之外，还贴心地提供了网页书签的导入功能，可以一键将你的网页书签导入进来，非常便利。
    如果你想将你的书签提供给别人，这里还有导出书签的选项，生成的HTML文件可以被主流浏览器所识别并完成导入。

    后台任务

    为了软件能够更好地运行，我创建了两个后台任务，一个用来进行提醒（和提醒类型挂钩），另一个用来自动备份。
    由于UWP机制限制，后台线程不允许一直保持监听，最低每15分钟唤醒一次。所以提醒类型的时间跨度设置为了15分钟，这意味着当你设置了提醒时，软件将会在你的预定时间提前15分钟左右提醒你。
    自动备份每天唤醒一次，按照你设置的备份周期进行自检，时间到了就会自动进行备份同步。如果你关闭了自动备份，该后台将注销。
