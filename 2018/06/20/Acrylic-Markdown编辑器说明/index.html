    欢迎使用 Acrylic Markdown 编辑阅读器

    人人皆有一颗写作心，我理解您需要更便捷更高效的工具记录你的所思所想，记录日常或知识，并将其中承载的价值传播给他人，Acrylic Markdown 是我献给您的礼物 —— 这是一款记录与和分享写作提供的专业工具。 您可以使用 Acrylic Markdown做以下事情：

    • 整理知识，学习笔记
    • 写作日记，杂文，所思所想
    • 撰写发布技术文稿（代码支持）

    请保留此份 Acrylic Markdown 的欢迎稿兼使用说明，如需撰写新稿件，点击左侧添加新的笔记 或者使用快捷键 Ctrl+N。

    什么是 Markdown

    Markdown 是一种方便记忆、书写的纯文本标记语言，用户可以使用这些标记符号以最小的输入代价生成极富表现力的文档：譬如您正在阅读的这份文档。它使用简单的符号标记不同的标题，分割不同的段落，粗体 或者 斜体 某些文字，更棒的是，它还可以

    1. 高亮一段代码[^code]

    2. 绘制表格

    商品 价格 数量
    电脑 ￥7000 1
    耳机 ￥800 2
    女友 NaN NaN

    3. 更详细语法说明

    想要查看更详细的语法说明，可以参考我为你准备的 Acrylic Markdown 语法手册，在左侧文件列表中可以查看。

    总而言之，不同于其它 所见即所得 的编辑器：你只需使用键盘专注于书写文本内容，就可以生成印刷级的排版格式，省却在键盘和工具栏之间来回切换，调整内容和格式的麻烦。Markdown 在流畅的书写和印刷级的阅读体验之间找到了平衡。 目前它已经成为世界上最大的技术分享网站 GitHub 和 技术问答网站 StackOverFlow 的御用书写格式。

    什么是 Acrylic Markdown

    您可以使用很多工具书写 Markdown，但是 Acrylic Markdown 应当是UWP平台上目前在UI设计/功能/解析度 上完成度最高的Markdown编辑工具，它具备了一个UWP应用天生的优点：轻量、触控友好、设计优秀，除此之外，它还具备了一些非常有趣的特性：

    1. 四档界面随心切换，明暗主题任君选择

    Acrylic Markdown 的主界面分作三区，从左到右依次为菜单栏、文档列表和编辑区。其中编辑区又可以调整为单页和双栏。如此，共计四种模式，你可以关闭菜单栏和文档列表的显示，并使用单页进行写作，享受完全沉浸式写作的乐趣，或者打开双栏，一边写一边预览更改，且双栏模式支持同步滚动哦~

    秉承优质设计的理念，Acrylic Markdown支持Light 和 Dark 两种主题，且两种主题均经过细心的调教，让您写得舒心，看得更舒心。

    2. 标签、分类和搜索

    Acrylic Markdown的文件管理比较特别，采用以分类为主，标签为辅的双线管理模式，简而言之就是支持分类检索和标签检索。

    除此之外，还提供了全部文件以创建时间倒序排列 和 最近打开以修改时间为倒序排列两种整体排序方式。

    搜索被放置在文件列表之中，输入文本后，任何标题内包含该文本的文档将会被筛选出来以供选择。

    3. 编辑工具栏与快捷键

    也许您对Markdown的语法还不甚熟悉，所以在你完全熟悉之前，我为您提供了一个如下图所示的工具栏，您可以先行在文本输入区域选中文字，再在工具栏中点击，以达致想要的效果。不过学好Markdown标记依然是一件非常重要的事哦~

    在您熟悉了Markdown语法之后，我还为您提供了一种更快速地写作方式，即快捷键。快捷键不仅涉及Markdown语法的快速添加，同时也涉及软件本身的操作。软件涉及的全部的快捷键我已单独抽出来放在了Acrylic Markdown快捷键列表中，如果您感兴趣，可以去看看，

    4. 管理工具栏

    为了便于管理您的文稿，在 编辑区 的底部放置了如下所示的 管理工具栏：

    通过管理工具栏可以：

    • : 保存文件
    • : 上传或下载云中的文件
    • : 将Markdown渲染成HTML并发送邮件
    • : 导出文件，支持以下格式：
      • -HTML
      • -Markdown
      • -图片
    • : 删除文件

    5. OneDrive图床与资源列表

    Acrylic Markdown自身集成了图床功能，所谓图床，简而言之就是将本地图片转化为网络图片，从而使Markdown文件的图片显示不再受本地路径约束，极大地增强了Markdown的可分享性。

    使用OneDrive图床需要你有一个微软账户，不过我相信你既然下载了软件，这对你肯定不是问题。那么在任意文档的编辑页面，点击工具栏中或即可激活OneDrive授权引导，遵从引导，并且按照软件说明完成配置工作，你就可以开始愉悦地使用OneDrive图床啦~

    值得一提的是，你所有的已上传资源会被软件记录下来，点击按钮就能看到你过往所上传的图片资源啦。此外，资源列表还内嵌了一个搜索模块，可以通过关键词的搜索找到你需要的图片。因此，在上传图片时，有一个良好的命名习惯是很必要的哦~

    6. 云同步

    如果您需要在不同的设备间编辑Markdown文档，Acrylic Markdown可以实现您的需求，您可以将单个Markdown文件上传至OneDrive云备份，或者在设置中将所有文档全部同步到OneDrive中。

    这样即使您换了电脑，文档也不会丢失，而是会在OneDrive中永久保留。

    当然了，如果你想在其它的移动设备中使用其他软件编辑Markdown文件，可能要稍微麻烦点了。因为在OneDrive中保存的文件并非是以你给定的标题命名，而是以文件ID做名。您可以解析OneDrive应用文件夹内的BackupFiles.json来获取标题与Id值之间的对应关系，这样可以帮助你找到需要的文档，但请注意，不要更改Markdown文件夹中的文件名，这会造成同步失败。

    7. 代码高亮

    借助CodeMirror的支持，得以以较小的性能代价实现实时的代码高亮，在此表示感谢。

    代码高亮颜色会随软件主题切换，晚上的颜色会比较骚一点。

    8. 自动保存

    软件支持自动保存文档，实现存储文档与输入文档的分离，便于软件进行文档管理。您可以在设置中设置自动保存的时间间隔。

    9. 阅读模式

    清除所有干扰，认真阅读渲染后的Markdown文件。

    仅在从外部打开Markdown文件时触发。如果在软件已经打开，则会弹出警告。

    你可以在设置中选择使用本地（XAML）渲染还是HTML渲染。

    10. 自定义渲染

    软件自带了一个CSS样式，名为Anatole，用于将Markdown渲染为HTML时进行样式加工。但是你也可以自行添加属于自己的CSS样式。

    你可以在设置中导入你的CSS样式表，或者对你当前的CSS备选列表进行操作。

    CSS渲染被应用在发送邮件、导出网页和阅读模式渲染之中
